Website Planning Document

Project Name: Healthy Family Recipes

HFR

Site Name

Healthy Family Recipes

I chose this name because it describes the purpose of the website: to provide healthier meal ideas and recipes for our family in a simple and practical way. The name is easy to remember and represents the content of the website.

Optional domain availability: healthyfamilyrecipes.org

Site Purpose

The purpose of this website is to provide an organized place where we can browse, add, and search for healthy recipes. The site will help our family prepare better meals at home by offering recipe suggestions, ingredients, and preparation instructions.

Scenarios

  • How can I find healthy recipes for dinner that my family will enjoy?
  • Where can I add a new recipe with ingredients and preparation steps?
  • Can I search for recipes by category, like breakfast, lunch, dinner, or dessert?

Color Schema

These are the main colors selected for the site.

#2E7D32

Used for headings, header background, and important titles.

#F4F1EA

Used for the page background to create a soft and clean look.

#FF7043

Used as an accent color for borders, highlights, and buttons.

Typography

Roboto will be used for the body text because it is clean, readable, and widely supported.

Roboto Bold will be used for headings and section titles to create emphasis and improve structure.

An optional accent font may be used later for recipe titles if needed, but the main font will remain simple for readability.
